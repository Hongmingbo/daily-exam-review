📌 一、必背公式（附适用条件，易混辨析）

① 速度公式
$v = \dfrac{s}{t}$
⚠️ 注意：速度单位要统一为m/s或km/h，不能混用！
② 欧姆定律
$I = \dfrac{U}{R}$，变形：$U = IR$，$R = \dfrac{U}{I}$
⚠️ 易错：电阻 $R$ 与电压、电流无关！不能认为"电压越大电阻越大"。
③ 功和功率
$W = Fs$（功 = 力 × 沿力方向移动的距离）
$P = \dfrac{W}{t} = Fv$
⚠️ 易错：$s$ 必须是沿力方向的分位移，不是总路程！
④ 机械效率
$\eta = \dfrac{W_{有用}}{W_{总}} imes 100\%$
⚠️ 易错：机械效率永远小于100%，永动机不存在！
⑤ 密度公式
$ ho = \dfrac{m}{V}$
⚠️ 密度是物质本身属性，与 $m$、$V$ 无关（但受温度影响）

📝 二、福建中考物理压轴题（精选3道）

【2024年福建中考物理·第25题（压轴）】

2024福建·25题

（1）小明用如图所示的电路测未知电阻 $R_x$ 的阻值。已知电源电压恒为 $6V$，$R_0 = 10\Omega$。当开关S闭合、S1断开时，电流表示数为 $0.3A$；当S、S1均闭合时，电流表示数为 $0.5A$。求 $R_x$ 的阻值。

（2）若将上述电路中的电源更换为电压可调的学生电源，保持 $R_0$ 不变，要求在不损坏元件的前提下，测量多组数据求平均值以减小误差。请设计两种不同的测量方案。

【答案】

（1）当S闭合、S1断开时：只有 $R_0$ 接入电路
$I_1 = 0.3A$，$U = I_1 R_0 = 0.3 imes 10 = 3V$
当S、S1均闭合时：$R_0$ 与 $R_x$ 并联，总电流 $I_2 = 0.5A$
通过 $R_x$ 的电流 $I_x = I_2 - I_1 = 0.5 - 0.3 = 0.2A$
$R_x = \dfrac{U}{I_x} = \dfrac{3}{0.2} = 15\Omega$
∴ $R_x = 15\Omega$

（2）方案一：电流表测总电流，改变电源电压，用 $R = \dfrac{U}{I}$ 计算。
方案二：使用滑动变阻器，改变接入电路的阻值，测得多组 $U$、$I$ 数据，作出 $U-I$ 图像，图像斜率即为 $R_x$（此方案可有效减小偶然误差）。

📌 题目分析：第（1）问考察串并联电路的电流分配关系，属于中档题；第（2）问考察测量电阻的多种方案设计，属于难题。易错点：在第（1）问中，当S1闭合时错误地认为电路中只有 $R_x$，而忽略了 $R_0$ 仍然在电路中。

【2023年福建中考物理·第23题（压轴）】

2023福建·23题

小明用一动滑轮提升重物（不计绳重和摩擦），已知物体重 $G=400N$，动滑轮重 $G_0=100N$。绳子自由端拉力为 $F$，物体被提升的高度为 $h$。（1）求拉力 $F$ 的大小和绳子自由端移动的距离 $s$ 与 $h$ 的关系。（2）若该装置的机械效率为 $80\%$，求实际拉力 $F$ 的大小。

【答案】

（1）$F = \dfrac{G + G_0}{2} = \dfrac{400+100}{2} = 250N$，$s = 2h$（动滑轮特点）

（2）$\eta = \dfrac{W_{有用}}{W_{总}} = \dfrac{Gh}{Fs}$，又 $s = 2h$，代入得 $\eta = \dfrac{G}{2F}$
$F = \dfrac{G}{2\eta} = \dfrac{400}{2 imes 0.8} = 250N$∴ 实际拉力 $F = 250N$

【2022年福建中考物理·第24题（压轴）】

2022福建·24题

一束光线从空气斜射入水中，入射角为 $45°$，折射角为 $32°$。（1）求水的折射率。（2）光从水射向空气时，使全反射发生的最小入射角（临界角）。

【答案】

（1）$n = \dfrac{\sin i}{\sin r} = \dfrac{\sin 45°}{\sin 32°} ≈ 1.33$

（2）由 $\sin C = \dfrac{1}{n}$，$C ≈ 48.8°$∴ 最小入射角约为 $48.8°$
