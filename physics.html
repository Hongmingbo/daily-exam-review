📌 必背公式（标注适用条件）

① 速度公式
v = s / t

适用条件：匀速直线运动

单位：v—m/s，s—m，t—s

注意：全程平均速度 ≠ 各段速度平均值

② 密度公式
ρ = m / V

适用条件：同种物质的均匀物体

单位：ρ—kg/m³，m—kg，V—m³

注意：密度是物质特性，与 m、V 无关

③ 压强公式
p = F / S 　　（固体压强）
p = ρgh 　　（液体压强）

固体压强适用条件：计算柱形固体对水平面的压强

液体压强适用条件：液体内部压强，h 为深度（液面到该点距离）

易错：液体压强只与 ρ液、h 有关，与液体重力、容器形状无关！

④ 欧姆定律
I = U / R

适用条件：纯电阻电路

导出公式：U = IR 　　R = U / I

注意：电阻是导体特性，与 U、I 无关

⑤ 电功与电功率
W = UIt
P = W / t = UI
P = I²R = U² / R 　　（纯电阻）

适用条件：所有电路（总公式）；后两个仅适用于纯电阻

易错：实际功率不是额定功率，注意区分！

📝 福建中考物理练习题

【电路分析 · 滑动变阻器】

小明设计了如图所示的电路，电源电压恒定，R₁为定值电阻，R₂为滑动变阻器，闭合开关S后：

(1) 当滑片P在a端时，电流表示数为0.3A，电压表示数为3V，求R₁的阻值；

(2) 当滑片P在b端时，电压表示数为6V，求此时电路消耗的总功率；

(3) 若电压表量程为0～3V，电流表量程为0～0.6A，为保护电表安全，滑动变阻器连入电路的阻值范围是多少？

【答案】

(1) 解：当滑片P在a端时，R₂被短路，只有R₁工作

R₁ = U₁ / I = 3V / 0.3A = 10Ω

(2) 解：当滑片P在b端时，设电源电压为U

U = U₁ + U₂ = 3V + 6V = 9V

由 I = U / (R₁ + R₂)，且 U₁ = IR₁ = (9V · 10Ω) / (10Ω + R₂) = 3V

解得：R₂ = 20Ω

P总 = UI = 9V × 0.3A = 2.7W

(3) 解：

电流表量程 0～0.6A：I_max = 0.6A

R₂_min = U / I_max − R₁ = 9V / 0.6A − 10Ω = 5Ω

电压表量程 0～3V：U₂_max = 3V

U₂ = UR₂ / (R₁ + R₂) ≤ 3V ⇒ 9R₂ ≤ 3(10 + R₂) ⇒ R₂ ≤ 5Ω

两种限制取交集，R₂ ≥ 5Ω，实际范围：5Ω ≤ R₂ ≤ 20Ω

⚠️ 易错点：电压表示数变化时，要分析是哪个量达到了限制！

【比热容 · 热学实验】

某实验小组测量物质的比热容，实验装置如图所示。

(1) 实验中加热时间越长，说明吸收的热量越____；

(2) 比较A、B两图，可得出的结论是____；

(3) 若液体甲的比热容为 c，质量为 m，温度从 t₁ 升高到 t₂，求它吸收的热量 Q = ____。

【答案】

(1) 多 / 大

(2) 质量相同的不同物质，升高相同温度，吸收的热量不同

(3) Q = cm(t₂ − t₁)

⚠️ 易错点：比热容是物质特性，与质量、体积无关！

【机械效率 · 滑轮组】

某工人用滑轮组提升重物，重物重G=400N，滑轮组机械效率80%。

(1) 求工人拉力F的大小；

(2) 若重物被提升2m，求有用功和总功。

【答案】

(1)

η = W有用 / W总 = Gh / Fs

F = Gh / (ηs) 　　（假设3段绳子，s = 3h = 6m）

F = (400N × 2m) / (0.8 × 6m) = 800 / 4.8 ≈ 167N

(2)

W有用 = Gh = 400N × 2m = 800J

W总 = W有用 / η = 800J / 0.8 = 1000J

⚠️ 易错点：机械效率 η < 1，总功 > 有用功！
