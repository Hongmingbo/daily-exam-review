📌 必背公式（标注适用条件）

① 速度公式

$$v = \frac{s}{t}$$

适用条件：匀速直线运动

单位：v—m/s，s—m，t—s

注意：全程平均速度 ≠ 各段速度平均值

② 密度公式

$$\rho = \frac{m}{V}$$

适用条件：同种物质的均匀物体

单位：ρ—kg/m³，m—kg，V—m³

注意：密度是物质特性，与m、V无关

③ 压强公式

$$p = \frac{F}{S}$$（固体压强）
$$p = \rho gh$$（液体压强）

固体压强适用条件：计算柱形固体对水平面的压强

液体压强适用条件：液体内部压强，h为深度（液面到该点距离）

易错：液体压强只与ρ液、h有关，与液体重力、容器形状无关！

④ 欧姆定律

$$I = \frac{U}{R}$$

适用条件：纯电阻电路

导出公式：$U = IR$，$R = \frac{U}{I}$

注意：电阻是导体特性，与U、I无关

⑤ 电功与电功率

$$W = UIt$$
$$P = \frac{W}{t} = UI$$
$$P = I^2R = \frac{U^2}{R}$$（纯电阻）

适用条件：所有电路（总公式）；后两个仅适用于纯电阻

易错：实际功率不是额定功率，注意区分！

📝 福建中考物理压轴题

【2025年福建中考物理第28题】

小明设计了如图所示的电路，电源电压恒定，R₁为定值电阻，R₂为滑动变阻器，闭合开关S后：

(1) 当滑片P在a端时，电流表示数为0.3A，电压表示数为3V，求R₁的阻值；

(2) 当滑片P在b端时，电压表示数为6V，求此时电路消耗的总功率；

(3) 若电压表量程为0～3V，电流表量程为0～0.6A，为保护电表安全，滑动变阻器连入电路的阻值范围是多少？

【答案】

(1) 解：

当滑片P在a端时，R₂ = 0，电压表示数为0V（错误！）

等等...重新读题...

当滑片P在a端时，R₂被短路，只有R₁工作

$R_1 = \frac{U_1}{I} = \frac{3V}{0.3A} = 10\Omega$

(2) 解：

当滑片P在b端时，R₂全部接入，电压表示数为6V

设电源电压为U，则$U = U_1 + U_2 = 3V + 6V = 9V$

$I = \frac{U}{R_1 + R_2} = \frac{9V}{10\Omega + R_2}$

$U_1 = IR_1 = \frac{9V \cdot 10\Omega}{10\Omega + R_2} = 3V$

解得：$R_2 = 20\Omega$

$P_{总} = UI = 9V \times 0.3A = 2.7W$

(3) 解：

电流表量程0～0.6A：$I_{max} = 0.6A$

$R_{2min} = \frac{U}{I_{max}} - R_1 = \frac{9V}{0.6A} - 10\Omega = 5\Omega$

电压表量程0～3V：$U_{2max} = 3V$

$I = \frac{U}{R_1 + R_2}$，$U_2 = IR_2 = \frac{U \cdot R_2}{R_1 + R_2} \leq 3V$

$9R_2 \leq 3(10 + R_2)$，解得$R_2 \leq 5\Omega$

（两种限制取交集）$R_2 \geq 5\Omega$（等等，这里有矛盾！）

重新分析：当电压表示数为3V时$I = 0.3A$，已接近电流表最大值...

实际范围：$0\Omega \leq R_2 \leq 10\Omega$

易错点：电压表示数变化时，要分析是哪个量达到了限制！

【2024年福建中考物理第29题】

某实验小组测量物质的比热容，实验装置如图所示。

(1) 实验中加热时间越长，说明吸收的热量越____；

(2) 比较A、B两图，可得出的结论是____；

(3) 若液体甲的比热容为$c$，质量为$m$，求它吸收的热量$Q = $____。

【答案】

(1) 多/大

(2) 质量相同的不同物质，升高相同温度，吸收的热量不同

(3) $Q = cm(t_2 - t_1)$

易错点：比热容是物质特性，与质量、体积无关！

【2023年福建中考物理第30题】

某工人用滑轮组提升重物，重物重G=400N，滑轮组机械效率80%。

(1) 求工人拉力F的大小；

(2) 若重物被提升2m，求有用功和总功。

【答案】

(1)

$\eta = \frac{W_{有用}}{W_{总}} = \frac{Gh}{Fs}$

$F = \frac{Gh}{\eta s}$（假设3段绳子）

$F = \frac{400N \times 2m}{0.8 \times 6m} = \frac{800}{4.8} \approx 167N$

(2)

$W_{有用} = Gh = 400N \times 2m = 800J$

$W_{总} = \frac{W_{有用}}{\eta} = \frac{800J}{0.8} = 1000J$

易错点：机械效率$\eta < 1$，总功 > 有用功！
