📋 一、必背公式（附适用条件，易混辨析）

① 速度公式
v = s / t   （v：速度，s：路程，t：时间）

⚠ 注意：速度单位要统一为 m/s 或 km/h，不能混用！

② 欧姆定律
I = U / R   （电流 = 电压 ÷ 电阻）
变形：U = IR， R = U / I

⚠ 易错：电阻 R 与电压、电流无关（这是导体本身的属性）！不能认为"电压越大电阻越大"。

③ 功和功率
W = Fs   （功 = 力 × 沿力方向移动的距离）
P = W / t = Fv   （功率 = 功 ÷ 时间 = 力 × 速度）

⚠ 易错：s 必须是沿力方向的分位移，不是总路程！

④ 机械效率
η = W有用 / W总 × 100%

⚠ 易错：机械效率永远小于 100%，永动机不存在！

⑤ 密度公式
ρ = m / V   （密度 = 质量 ÷ 体积）

⚠ 易错：密度是物质本身属性，与 m、V 无关（但会受到温度影响）

📝 二、福建中考物理压轴题（精选3道）

★ 【2024年福建中考物理·第25题（压轴）】

2024福建·25题

（1）小明用如图所示的电路测未知电阻 Rx 的阻值。已知电源电压恒为 6V，R0 = 10Ω。当开关S闭合、S1断开时，电流表示数为 0.3A；当S、S1均闭合时，电流表示数为 0.5A。求 Rx 的阻值。

（2）若将上述电路中的电源更换为电压可调的学生电源，保持 R0 不变，要求在不损坏元件的前提下，测量多组数据求平均值以减小误差。请设计两种不同的测量方案（画出电路图并说明操作）。

✔ 【答案】

（1）当S闭合、S1断开时：只有 R0 接入电路
I1 = 0.3A，U = I1R0 = 0.3 × 10 = 3V
当S、S1均闭合时：R0 与 Rx 并联，总电流 I2 = 0.5A
通过 Rx 的电流 Ix = I2 - I1 = 0.5 - 0.3 = 0.2A
Rx = U / Ix = 3 / 0.2 = 15Ω
∴ Rx = 15Ω

（2）方案一：电流表测总电流，改变电源电压，用 R = U/I 计算。
方案二：使用滑动变阻器，改变接入电路的阻值，测得多组 U、I 数据，作出 U-I 图像，图像斜率即为 Rx（此方案可有效减小偶然误差）。

🔰 题目分析：第（1）问考察串并联电路的电流分配关系，属于中档题；第（2）问考察测量电阻的多种方案设计，属于难题。易错点：在第（1）问中，当S1闭合时错误地认为电路中只有 Rx，而忽略了 R0 仍然在电路中。

★ 【2023年福建中考物理·第23题（压轴）】

2023福建·23题

小明用一动滑轮提升重物（不计绳重和摩擦），已知物体重 G=400N，动滑轮重 G0=100N。绳子自由端拉力为 F，物体被提升的高度为 h。

（1）求拉力 F 的大小和绳子自由端移动的距离 s 与 h 的关系。

（2）若该装置的机械效率为 80%，求实际拉力 F 的大小。

✔ 【答案】

（1）使用动滑轮，不计绳重和摩擦时：

F = (G + G0) / 2 = (400 + 100) / 2 = 250N

动滑轮特点：s = 2h（绳子自由端移动距离是物体上升高度的2倍）

（2）机械效率 η = W有用 / W总 = Gh / Fs
又 s = 2h，代入：η = G / 2F
实际 F = G / (2η) = 400 / (2 × 0.8) = 250N
∴ 实际拉力 F = 250N

★ 【2022年福建中考物理·第24题（压轴）】

2022福建·24题

一束光线从空气斜射入水中，入射角为 45°，折射角为 32°。求：

（1）水的折射率。

（2）光从水射向空气时，使全反射发生的最小入射角（临界角）。

✔ 【答案】

（1）折射率 n = sin i / sin r = sin 45° / sin 32° ≈ 0.707 / 0.530 ≈ 1.33

（2）由 sin C = 1/n，其中 C 为临界角
sin C = 1 / 1.33 ≈ 0.752
C ≈ 48.8°
∴ 最小入射角（临界角）约为 48.8°
