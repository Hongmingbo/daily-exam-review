📐 数学 · 急救包

① 勾股定理（必考）

公式：a² + b² = c²

常用勾股数：3-4-5、5-12-13、6-8-10、7-24-25、8-15-17

逆定理：三边满足 a²+b²=c² → 直角三角形

⚠️ 只适用于直角三角形！先判断直角再套公式

② 一元二次方程

求根公式：x = (-b ± √(b²-4ac)) / 2a

判别式：Δ = b²-4ac

Δ>0 两个不等实根 | Δ=0 两个相等实根 | Δ<0 无实根

韦达定理：x₁+x₂ = -b/a   x₁·x₂ = c/a

⚠️ 先化为标准型 ax²+bx+c=0 再用公式

③ 圆

弧长：l = nπr/180   扇形面积：S = nπr²/360 = lr/2

垂径定理：垂直于弦的直径平分弦，并平分弦所对的两条弧

圆内接四边形对角互补

④ 三角函数（特殊角）

sin²α + cos²α = 1

sin30° = cos60° = ½   cos30° = sin60° = √3/2

tan30° = √3/3   tan45° = 1   tan60° = √3

🚨 高频易错点

  • 分式方程一定要验根（去分母可能产生增根）
  • 二次函数顶点坐标 (-b/2a, (4ac-b²)/4a)
  • 概率题：有放回/无放回概率不同！
  • 相似三角形面积比 = 相似比的平方

⚡ 物理 · 急救包

① 凸透镜成像规律

u > 2f：成倒立缩小实像（照相机原理）

f < u < 2f：成倒立放大实像（投影仪原理）

u < f：成正立放大虚像（放大镜原理）

口诀：一倍焦距分虚实，二倍焦距分大小

② 电路分析

欧姆定律：I = U/R   →   U = IR   R = U/I

串联：I相等，U=U₁+U₂，R=R₁+R₂（分压）

并联：U相等，I=I₁+I₂，1/R=1/R₁+1/R₂（分流）

⚠️ 短路：电流不经过用电器，I急剧增大！

③ 机械功与功率

功：W = Fs（同一方向）

功率：P = W/t = Fv

机械效率：η = W有/W总 × 100%

动能：Eκ = ½mv²   重力势能：Eρ = mgh

🚨 高频易错点

  • 比热容公式：Q = cm(t₂-t₁)，c是物质的特性，与质量无关！
  • 速度单位换算：1m/s = 3.6km/h
  • 滑动摩擦力与速度无关，只与正压力和粗糙程度有关
  • 杠杆平衡：F₁L₁ = F₂L₂，L是力到支点的垂直距离

🧪 化学 · 急救包

① 酸碱盐反应

酸的通性：酸 + 金属 → 盐 + 氢气（排在H前的金属）

酸 + 碱 → 盐 + 水（中和反应）

酸 + 盐 → 新酸 + 新盐（符合复分解条件）

碱的通性：碱 + 非金属氧化物 → 盐 + 水

② 复分解反应条件

生成物中有↓（沉淀）或↑（气体）或H₂O（水）

常见沉淀：BaSO₄、AgCl、CaCO₃、Cu(OH)₂（蓝）、Fe(OH)₃（红褐）

③ 质量守恒定律

化学反应前后：各原子种类、数目、质量不变

化学方程式配平：观察法、最小公倍数法

④ 溶液配制

溶质质量分数：w = m溶质 / m溶液 × 100%

m溶液 = m溶剂 + m溶质 = ρV

🚨 高频易错点

  • 燃烧必须有O₂参与；缓慢氧化无火焰（如铁生锈）
  • 金属活动性顺序：K Ca Na Mg Al Zn Fe Sn Pb H Cu Hg Ag Pt Au
  • 置换反应：单质+化合物→新单质+新化合物
  • CO₂使澄清石灰水变浑浊，但SO₂也会，不能混淆

📝 语文 · 急救包

① 文言文高频实词

之：的（取消句子独立性）/ 代词（他、它）/ 动词（去、到）

其：他的 / 它的 / 加强语气（其真无马邪？）

而：表顺承（并且）/ 表转折（但是）/ 表修饰

以：用 / 因为 / 凭借 / 认为

于：在 / 向 / 对 / 比 / 从

② 作文万能素材（时事类）

航天精神：嫦娥六号月背采样 / 可用于：坚持、创新、强国梦

科技自立：芯片突破 / 可用于：自主研发、面对封锁

乡村振兴：脱贫攻坚成果 / 可用于：奋斗、民生、共同富裕

文化传承：非遗活化 / 可用于：传统与创新、文化自信

③ 病句类型

  • 成分残缺：缺少主语/谓语/宾语
  • 搭配不当：动宾不匹配、主谓不搭
  • 语序不当：多重定语/状语顺序错误
  • 重复累赘：意思相同的词重复出现
  • 不合逻辑：自相矛盾或不符合常识

🚨 答题规范

  • 赏析句子：手法 + 内容 + 情感 + 效果
  • 概括内容：人+事+结果，分点作答
  • 说明文：先找说明方法，再分析作用

📖 英语 · 急救包

① 完形填空高频短语

look forward to doing   look up at   look after = take care of

take part in = join in   set up = establish

turn off > turn on / turn down / turn up / turn in

pick up   run out (of)   depend on = rely on

make progress   make a decision   make sure

have something in common   have trouble (in) doing

② 动词时态判断

一般现在时：often/always/usually + 客观事实

现在进行时：now/look/listen + be + v-ing

一般过去时：yesterday/last week/ago + 动词过去式

现在完成时：already/yet/ever/never + have/has + 过去分词

过去进行时：at that time/while + was/were + v-ing

一般将来时：tomorrow/next week + will + 动词原形

③ 阅读理解技巧

  • 先读题目，划关键词，再回文章定位
  • 细节题在原文中找原句，不要靠记忆推断
  • 推理题看上下文，排除无中生有和过度推断项
  • 主旨题看首段和末段，关注反复出现的词

🚨 作文常用句型

There is something wrong with...

Whether...or not...   Not only...but also...

The + 比较级..., the + 比较级...   I believe...

🏛️ 历史 · 急救包

1919 五四运动 → 马克思主义传播，新民主主义开端

1921 中国共产党成立

② 世界史高频考点

文艺复兴（14-17世纪）：人文主义，意大利→欧洲

启蒙运动（18世纪）：理性主义，孟德斯鸠、伏尔泰、卢梭

工业革命（18世纪60年代英国）：瓦特改良蒸汽机

法国大革命（1789）：攻占巴士底狱 → 《人权宣言》

  • 意义题：性质+内容+地位+启示

🎯 政治 · 急救包

① 公民基本权利

人身自由：不受非法拘禁、搜查、逮捕

人格尊严：名誉权、肖像权、姓名权、隐私权

财产权利：占有权、使用权、收益权、处分权

监督权：对国家机关及工作人员批评、建议、申诉、控告

选举权与被选举权：年满18周岁，未被剥夺政治权利

② 改革开放成就

经济：GDP世界第二，全面小康，综合国力增强

科技：高铁、5G、载人航天、杂交水稻

外交：人类命运共同体，"一带一路"

启示：坚持党的领导+改革开放+科学发展

③ 社会主义核心价值观

国家层面：富强、民主、文明、和谐

社会层面：自由、平等、公正、法治

个人层面：爱国、敬业、诚信、友善

🚨 答题模板

  • 原因/意义：国家+社会+个人，三层次作答
  • 做法类：从国家/企业/个人角度分别提
  • 认识类：是什么→为什么→怎么做
  • 环保类：保护环境+可持续发展+绿色发展
