📌 考点速记：10个易错化学式和概念

1. 系数与角标的区别

易错：2O 表示2个氧原子；O₂ 表示1个氧分子（含2个氧原子）

正确理解：系数表示分子个数，角标表示1个分子中原子的种类和数目

2. 铁的燃烧产物

易错：3Fe+2O₂=Fe₃O₄ 写成 2Fe+2O₂=2FeO

正确：铁在氧气中燃烧生成 Fe₃O₄（四氧化三铁），不是 FeO

3. 一氧化碳与二氧化碳

易错：CO 和 CO₂ 混淆，碳燃烧不充分产物写成 CO₂

正确：C + O₂（充足）= CO₂；2C + O₂（不充足）= 2CO

4. 实验室制氧气三大途径

易错：记混药品和反应条件

正确：

  • 2KMnO₄ = K₂MnO₄ + MnO₂ + O₂↑（高锰酸钾，加热）
  • 2KClO₃ = 2KCl + 3O₂↑（氯酸钾，二氧化锰催化，加热）
  • 2H₂O₂ = 2H₂O + O₂↑（过氧化氢，二氧化锰催化，常温）

5. 质量守恒定律

易错：认为反应前后分子总数不变，或忽略气体体积

正确：化学反应前后，原子种类、数目、质量不变，分子总数可能改变

6. 置换反应条件

易错：Zn + H₂SO₄ = ZnSO₄ + H₂↑ 写成 Zn + H₂SO₄ = ZnS + H₂O

正确：金属与酸反应，金属置换出氢，生成盐和水（注意不是直接生成金属硫化物）

7. 复分解反应条件

易错：认为所有酸碱盐混合都能反应

正确：必须有沉淀、气体或水生成才反应（生成物之一必须是沉淀、气体或弱电解质）

8. 氢氧化钙与二氧化碳反应

易错：Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O 写成 Ca(OH)₂ + CO₂ = CaHCO₃

正确：澄清石灰水变浑浊的原因，生成碳酸钙沉淀

9. 金属活动性顺序

易错：Mg + AlCl₃ = Al + MgCl₂（错，Mg没Al活泼，不能置换）

正确：金属活动性顺序 K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

10. pH 值与酸碱性

易错：pH=7 为中性，pH<7 为酸性，pH>7 为碱性；pH 越小酸性越强

正确：pH=7 中性；pH<7 酸性，且数值越小酸性越强；pH>7 碱性，数值越大碱性越强

📝 真题训练：福建中考化学压轴题（2023-2025）

【2025年福建中考·化学·压轴】

题目：某化学兴趣小组为了探究"酸碱中和反应"，进行了以下实验：

取一定量的稀盐酸于烧杯中，逐滴加入 NaOH 溶液，并用 pH 传感器测定溶液的 pH 变化，绘制曲线如图所示（图像描述：起点 pH≈1，终点 pH≈13，拐点 pH=7）。请回答：

  1. 稀盐酸与 NaOH 溶液反应的化学方程式是 _______________。
  2. 图中 a 点时，溶液中的溶质是 _______________（写化学式）。
  3. 当滴加 NaOH 溶液至 pH=7 时，所得溶液中的离子有 _______________（写离子符号）。
  4. 下列物质中，能与反应后的溶液发生反应的是 _______________（填序号）。
    ① Mg ② Na₂CO₃ ③ BaCl₂ ④ AgNO₃
【答案与解析】

答案：

  1. HCl + NaOH = NaCl + H₂O
  2. a 点时 pH<7，酸过量，溶质为 HCl 和 NaCl（化学式：HCl、NaCl）
  3. pH=7 时，酸碱恰好完全反应，溶质为 NaCl，离子有 Na⁺ 和 Cl⁻
  4. ③④（NaCl 可与 BaCl₂ 不反应（无沉淀气体水），与 AgNO₃ 反应生成 AgCl 沉淀）

【坑点】：① Mg 只与酸反应，pH=7 时无酸剩余；② Na₂CO₃ 与 NaCl 不反应；③④正确，AgNO₃ 与 NaCl 生成 AgCl 沉淀，BaCl₂ 与 NaCl 不反应。

【2024年福建中考·化学·压轴】

题目：有一包白色粉末，可能含有 NaCl、Na₂CO₃、Na₂SO₄、BaCl₂ 中的一种或几种。化学兴趣小组进行如下实验：

Ⅰ. 取少量粉末于试管中，加足量水溶解，过滤，得到无色滤液和白色沉淀 A；
Ⅱ. 向滤液中加入足量稀硝酸，再加入 AgNO₃ 溶液，产生白色沉淀 B；
Ⅲ. 向沉淀 A 中加入足量稀硝酸，沉淀部分溶解，并产生无色气体。

请回答：

  1. 白色沉淀 A 是 _______________。
  2. 原粉末中一定含有的物质是 _______________，一定不含有的物质是 _______________。
  3. 实验Ⅱ中生成白色沉淀 B 的化学方程式是 _______________。
【答案与解析】

答案：

  1. 沉淀 A 为 BaSO₄ 和 BaCO₃ 的混合物（部分溶于稀硝酸说明有 BaCO₃，不溶部分为 BaSO₄）
  2. 一定含有：Na₂CO₃、Na₂SO₄、BaCl₂；一定不含：无法确定 NaCl（因为 AgNO₃ 与 Cl⁻ 产生的 AgCl 会干扰，但 AgNO₃ 是在加稀硝酸后加入，所以 Ag⁺ 只与 Cl⁻ 结合生成 AgCl，故 NaCl 可能存在）→ 答案：一定有 Na₂CO₃、Na₂SO₄、BaCl₂；一定无 Na₂SO₄ 与 BaCl₂ 都会生成 BaSO₄ 沉淀... 解析：Ⅲ中沉淀部分溶解说明有 BaCO₃（溶于酸）和 BaSO₄（不溶酸），因此一定含 Na₂CO₃ 和 Na₂SO₄；能与 Na₂CO₃ 和 Na₂SO₄ 都生成沉淀的只有 BaCl₂，所以一定有 BaCl₂；NaCl 无法确定。
  3. AgNO₃ + NaCl = AgCl↓ + NaNO₃

【坑点】：Ⅲ中"部分溶解"是关键词，BaCO₃ 溶于酸而 BaSO₄ 不溶是解题关键。

【2023年福建中考·化学·压轴】

题目：为测定某铁合金中铁的质量分数，取 10g 铁合金样品，加入足量稀硫酸，充分反应后测得生成氢气 0.3g。求：

  1. 计算该铁合金中铁的质量分数。
  2. 若将同等质量的铁合金与足量硫酸铜溶液反应，理论上可生成铜的质量是多少？（写出计算过程）

已知：Fe + H₂SO₄ = FeSO₄ + H₂↑；Fe + CuSO₄ = FeSO₄ + Cu

【答案与解析】

答案：

（1）铁的质量分数：

解：设铁合金中铁的质量为 x

Fe + H₂SO₄ = FeSO₄ + H₂↑

56.................2

x..................0.3g

56/x = 2/0.3g

x = 56 × 0.3g / 2 = 8.4g

铁的质量分数 = 8.4g / 10g × 100% = 84%

（2）生成铜的质量：

解：设生成铜的质量为 y

Fe + CuSO₄ = FeSO₄ + Cu

56................64

8.4g..............y

56/8.4g = 64/y

y = 64 × 8.4g / 56 = 9.6g

答：该铁合金中铁的质量分数为 84%；理论上可生成铜 9.6g。

【坑点】：① 第二问用的是铁合金中铁的质量（8.4g），不是合金总质量；② 注意单位对应；③ 铁与硫酸铜反应是置换反应，铁是还原剂。

💡 今日复习建议

  • 重点突破：化学方程式书写是中考必考，重点练习配平方法（最小公倍数法、奇数配偶法）
  • 易错提醒：复分解反应条件（生成沉淀/气体/水）必须同时满足三个条件之一；置换反应注意金属活动性顺序
  • 图像题：pH-时间曲线题是高频压轴题，先找起点（pH<7 酸过量）、拐点（pH=7 恰好完全反应）、终点（pH>7 碱过量）
  • 计算题：质量守恒定律和化学方程式计算是送分题，确保格式规范（设-写-比-算-答）
  • 实验题：除杂题遵循"不引新杂质、不减原物质"原则，检验顺序很重要
