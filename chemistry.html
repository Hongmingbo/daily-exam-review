2026年5月16日（周六）

📌 考点速记：10个易错化学式和概念

1. 常见易错化学式
❌ 错误：CO2（一氧化碳）→ ✅ 正确：CO（一氧化碳），CO₂（二氧化碳）
❌ 错误：H2O2（过氧化氢）→ ✅ 正确：记住"双氧水"是H₂O₂
❌ 错误：NaCO3（碳酸钠）→ ✅ 正确：Na₂CO₃（纯碱）
❌ 错误：Ca(OH)2（氢氧化钙）→ ✅ 正确：熟石灰，Ca²⁺要写在前面
2. 化学反应基本类型
化合反应：A + B → AB（多变一）
分解反应：AB → A + B（一变多）
置换反应：A + BC → AC + B（单换单）
复分解反应：AB + CD → AD + CB（互换离子）
❌ 注意：复分解反应发生条件：生成物中有沉淀、气体或水
3. 质量守恒定律
六不变：原子种类、数目、质量不变；元素种类、质量不变；物质总质量不变
两变：分子种类一定变；物质种类一定变
❌ 易错：反应前后原子个数相等（配平就是保证这个）
4. 溶液配制
溶质质量分数 = 溶质质量 ÷ 溶液质量 × 100%
溶液质量 = 体积 × 密度（注意单位统一）
❌ 易错：稀释问题：稀溶液质量 = 浓溶液质量 + 加水量
5. 燃烧与灭火
燃烧条件：①可燃物；②与O₂接触；③达到着火点
灭火原理：破坏任一条件即可
❌ 易混：着火点是可燃物固有属性，不能降低，只能移除可燃物或隔绝O₂

📝 真题训练：3道福建中考化学压轴题

【压轴题1·2024福建中考】实验室制取CO₂并验证其性质：
（1）实验室制CO₂的化学方程式：___________________________
（2）选用↑↑↑制取装置（填"A""B"或"C"），理由是_________________
（3）将CO₂通入澄清石灰水，石灰水变浑浊，反应的化学方程式为：_____________
（4）若要得到干燥的CO₂，需要将气体通过_____________（填试剂名称）
✅ 参考答案：
（1）CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
（2）B或C（固液不加热）；反应物为大理石（块状）和稀盐酸（液体），不需加热
（3）CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
（4）浓硫酸（干燥CO₂）
🚫 坑点：不能用Ca(OH)₂干燥CO₂，会反应；也不能用CaCO₃（本身会反应）
【压轴题2·2023福建中考】关于金属活动性顺序：
（1）将铁片放入CuSO₄溶液中，现象：________________，化学方程式：_____________
（2）将铜片放入AgNO₃溶液中，现象：________________，化学方程式：_____________
（3）将锌片放入FeSO₄溶液中，反应能否发生？______（填"能"或"不能"），依据是：______
✅ 参考答案：
（1）铁片表面有红色固体析出，溶液由蓝色变浅绿色；Fe + CuSO₄ → FeSO₄ + Cu
（2）铜片表面有银白色固体析出；Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag
（3）能；Zn的金属活动性比Fe强，能将Fe从其盐溶液中置换出来
🚫 坑点：FeSO₄溶液是浅绿色，不是无色；CuSO₄溶液才是蓝色
【压轴题3·2022福建中考】计算题（溶液配制）：
要配制100g 9%的氯化钠溶液，需要NaCl_____g，水_____mL。配制时若用量筒量取水时仰视读数，则所配溶液质量分数会偏___________（填"大"或"小"）
✅ 参考答案：
9g；91mL；仰视读数使实际水量偏大，故质量分数偏小
🚫 坑点：仰视→读数比实际小→水加多了→质量分数偏小；俯视则相反

💡 复习建议

  • 化学式：每天默写5个化学式，注意元素符号大小写（Co≠CO）
  • 方程式：配平后检查原子个数是否守恒，这是计算题必检步骤
  • 溶液配制：仰视俯视读数是高频考点，画图理解最有效
  • 金属活动性：记口诀"钾钙钠镁铝，锌铁锡铅氢"，强换弱（活泼换不活泼）
  • 推断题：抓特征反应（生成气体、沉淀、颜色变化）作为突破口
