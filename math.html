📌 一、必背公式与概念（易混淆高频考点）

① 勾股定理

在直角三角形中：a² + b² = c²

其中 c 为斜边，a、b 为两直角边

⚠️ 易错：不是 a+b=c，而是 a²+b²=c²！

② 乘法公式（完全平方、平方差）

(a+b)² = a² + 2ab + b²

(a-b)² = a² - 2ab + b²

a² - b² = (a+b)(a-b)

⚠️ 易错：(a+b)² ≠ a² + b²（少了 2ab）

③ 一元二次方程求根公式

x = [ -b ± √(b² - 4ac) ] / 2a

（其中 Δ = b² - 4ac ≥ 0）

⚠️ 易错：分子是 -b ± √... 不是 b ±！

📝 二、福建中考数学压轴题

【2024年福建中考数学·第25题（压轴题）】

2024福建·25题

（1）已知二次函数 y = x² + bx + c 的图像经过点 (1,0) 和 (3,0)，求 b 和 c 的值。

（2）若（1）中的二次函数图像与 x 轴交于 A、B 两点（A 在左，B 在右），顶点为 C。求 △ABC 的面积。

（3）设 P 为抛物线上一点，且 S(△PAB) = 2 × S(△ABC)，求点 P 的坐标。

【答案】

（1）代入点 (1,0)：1 + b + c = 0 …… ①

代入点 (3,0)：9 + 3b + c = 0 …… ②

② - ① 得：8 + 2b = 0，解得 b = -4

代入①：1 - 4 + c = 0，解得 c = 3

∴ b = -4，c = 3

（2）函数为 y = x² - 4x + 3 = (x-2)² - 1

顶点 C(2, -1)，与 x 轴交点：A(1,0)、B(3,0)

△ABC 底 AB = 2，高 = |-1| = 1

S(△ABC) = (1/2) × 2 × 1 = 1

∴ △ABC 的面积为 1

（3）设 P(x, x²-4x+3)，则 △PAB 底为 AB=2，高为 |yP| = |x²-4x+3|

S(△PAB) = (1/2) × 2 × |x²-4x+3| = |x²-4x+3|

由题意：|x²-4x+3| = 2 × 1 = 2

解得：x = 2 + √3 或 x = 2 - √3

∴ P(2+√3, -1) 或 P(2-√3, -1)

📌 题目分析：第（1）问送分；第（2）问需要正确求出顶点坐标和三角形面积；第（3）问的难点在于理解 |x²-4x+3| 表示点 P 到 x 轴的距离，易错点是忘记取绝对值导致漏解。

📝 三、福建中考数学压轴题（2023年精选）

【2023年福建中考数学·第25题（压轴题）】

2023福建·25题

（1）在Rt三角形ABC中，角C=90度，AC=6，BC=8。D是BC的中点，E是AB上一点，且DE垂直于AB。求DE的长。

（2）在（1）的条件下，若F是AC上一点，且满足EF//BC，求三角形DEF的面积。

（3）若G是三角形ABC内部一点，且满足GD//AC，GE//BC，GF//AB，求点G到三角形ABC三边距离之和的最小值。

【答案】

（1）在Rt三角形ABC中，AC=6，BC=8，由勾股定理：AB = √(6² + 8²) = √(36+64) = 10

D是BC中点，CD = BD = 4

三角形BCD面积 = (1/2) × CD × AC = (1/2) × 4 × 6 = 12

又三角形BCD面积 = (1/2) × AB × DE = (1/2) × 10 × DE

即 (1/2) × 10 × DE = 12，解得 DE = 12/5 = 2.4

（2）EF//BC，三角形AEF与三角形ABC相似（AA相似）

由相似三角形性质：AE/AB = AF/AC = EF/BC

代入 AB=10，AC=6：设 AE = x，则 AF = (6/10)x = (3/5)x

第（1）问已得 DE = 12/5，即三角形ADE的高为 12/5

设 E 在 AB 上，由坐标法或合速度三角形可得 AE = 6

验证：AE/AB = 6/10 = 3/5，对应 EF/BC = 3/5，故相似比为 3/5

S(DEF) / S(ABC) = (3/5)² = 9/25

S(ABC) = (1/2) × 6 × 8 = 24

故 S(DEF) = 24 × (9/25) = 72/25

（3）由平行条件，G为三角形ABC的"重心型"分割点

设G到BC、AC、AB的距离分别为d1、d2、d3，由平行条件：d1 : d2 : d3 = 1 : 2 : 3

三角形ABC面积 = (1/2)×6×8 = 24 = (1/2)(8d1 + 6d2 + 10d3)

设 d1 = k，则 d2 = 2k，d3 = 3k，代入得 (1/2)(8k+12k+30k)=24

解得 k = 24/25，d1 + d2 + d3 = 6k = 144/25

📌 题目分析：第（1）问考察勾股定理和等面积法，是基础题；第（2）问考察平行线分线段成比例；第（3）问难度最大，涉及几何最值，难点在于建立距离比例关系并利用面积恒等式求解。
