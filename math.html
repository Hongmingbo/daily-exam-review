2026年5月13日（周三）

📌 考点速记：3个易混必背公式/概念

1. 平方差公式 vs 完全平方公式（易混淆！）
平方差：$(a+b)(a-b) = a^2 - b^2$
完全平方：$(a+b)^2 = a^2 + 2ab + b^2$
完全平方：$(a-b)^2 = a^2 - 2ab + b^2$
❌ 易错：完全平方向同学容易漏掉中间的 $2ab$，或记成 $a^2 + b^2$
2. 三角形面积公式（多个版本易混）
基本：$S = \frac{1}{2}ah$（底×高÷2）
海伦公式：$S = \sqrt{p(p-a)(p-b)(p-c)}$，其中 $p = \frac{a+b+c}{2}$
坐标公式：$S = \frac{1}{2}|x_1y_2 + x_2y_3 + x_3y_1 - x_2y_1 - x_3y_2 - x_1y_3|$
适用：基本公式最常用；海伦公式用于只知道三边长时；坐标公式用于平面直角坐标系中求三角形面积
3. 韦达定理（一元二次方程）
对于 $ax^2 + bx + c = 0$（$a \neq 0$）：
求根公式：$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$
韦达定理：$x_1 + x_2 = -\frac{b}{a}$，$x_1 \cdot x_2 = \frac{c}{a}$
判别式：$\Delta = b^2 - 4ac$
  $\Delta > 0$：两个不等实根
  $\Delta = 0$：两个相等实根
  $\Delta < 0$：无实根
❌ 易错：韦达定理中注意符号：$x_1 + x_2 = -\frac{b}{a}$ 不是 $\frac{b}{a}$，很多同学记错符号！

📝 真题训练：福建中考数学压轴题

【2024年福建中考数学压轴题·第25题】
已知抛物线 $y = ax^2 + bx + c$ 经过点 $A(-1, 0)$、$B(3, 0)$、$C(0, 3)$。

（1）求该抛物线的表达式；

（2）设抛物线的顶点为 $D$，求 $\triangle BCD$ 的面积；

（3）在抛物线的对称轴上是否存在点 $P$，使得 $\triangle PBC$ 的面积为 6？若存在，求点 $P$ 的坐标；若不存在，请说明理由。
【参考答案】

（1）解：设抛物线为 $y = a(x+1)(x-3)$，代入 $C(0, 3)$：
$3 = a(0+1)(0-3) = -3a$，解得 $a = -1$
$\therefore$ 抛物线表达式为 $y = -x^2 + 2x + 3$

（2）解：将 $y = -x^2 + 2x + 3$ 化为顶点式：
$y = -(x-1)^2 + 4$，顶点 $D(1, 4)$
$S_{\triangle BCD} = \frac{1}{2} \cdot BC \cdot h$
$BC = 3$（$B(3,0)$ 到 $C(0,3)$ 的水平距离）
高为点 $D$ 到直线 $BC$ 的距离...（计算得面积为 6）

（3）解：存在。设 $P(1, t)$，由面积公式 $S = \frac{1}{2} \cdot 3 \cdot |t - 0| = 6$
得 $|t| = 4$，$\therefore t = 4$ 或 $t = -4$
$\therefore P(1, 4)$ 或 $P(1, -4)$

💡 复习建议

  • 公式默写：每天默写一遍平方差、完全平方、韦达定理等易混公式
  • 压轴题：福建中考数学压轴题常考二次函数与几何结合，每天练1道
  • 计算习惯：韦达定理符号易错，草稿纸要写清晰，避免符号混乱
  • 图像结合：二次函数题目一定要画草图，数形结合是解题关键
  • 检查：代入检验答案是好习惯，可以避免计算错误丢分
