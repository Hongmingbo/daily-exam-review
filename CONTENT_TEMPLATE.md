## 化学篇（每周三）

```markdown
## 化学 · 知识点复习

### 📌 物质构成

#### ① 元素符号速记
H He Li Be B C N O F Ne Na Mg Al Si P S Cl Ar K Ca...

#### ② 化学式书写
- 化合价口诀：一价钾钠氯氢银，二价氧钙钡镁锌...

### ⚗️ 化学反应

#### ① 质量守恒定律
- 六不变：原子种类/数目/质量不变；元素种类/质量不变；物质总质量不变
- 两变：分子种类一定变；原子组合一定变

#### ② 四大基本反应类型
| 类型 | 特点 | 通式 |
|------|------|------|
| 化合反应 | 多变一 | A+B→AB |
| 分解反应 | 一变多 | AB→A+B |
| 置换反应 | 单换单 | A+BC→AC+B |
| 复分解反应 | 互相交换 | AB+CD→AD+CB |

### 🧪 酸碱盐

#### 酸的通性
- 与指示剂反应：石蕊变红/酚酞不变
- 与金属反应：Fe+H₂SO₄→FeSO₄+H₂↑
- 与碱反应：HCl+NaOH→NaCl+H₂O

### 💡 计算专题
1. 化学方程式：写→配→标→算
2. 质量分数：某元素质量分数 = (原子数×Ar) / Mr × 100%
```

## 提交流程

```bash
# 1. 新建内容文件
cp CONTENT_TEMPLATE.md content/2026-06-01-math.md
# 编辑内容...

# 2. 运行同步脚本（IMA KB → 网站）
python3 scripts/kb_sync.py

# 3. 推送部署
git add . && git commit -m "更新 2026-06-01 数学内容" && git push
```
