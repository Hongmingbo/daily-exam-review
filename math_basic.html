📖 公式定理 · 数学

📌 中考必背核心公式

① 勾股定理（8年级下册，中考高频）

公式：a² + b² = c²

常用勾股数：3-4-5、5-12-13、6-8-10、7-24-25、8-15-17

逆定理：若三边满足 a² + b² = c²，则为直角三角形

易错：勾股定理只适用于直角三角形，先判断直角

② 一元二次方程

求根公式：x = (-b ± √(b² - 4ac)) / 2a

判别式：Δ = b² - 4ac

Δ > 0：两个不等实根   Δ = 0：两个相等实根   Δ < 0：无实根

韦达定理（福建高频）：x₁ + x₂ = -b/a   x₁ · x₂ = c/a

③ 圆

弧长：l = nπr / 180   扇形面积：S = nπr² / 360 = lr / 2

垂径定理：垂直于弦的直径平分弦，并平分弦所对的两条弧

圆内接四边形对角互补

④ 三角函数

sin²α + cos²α = 1

sin30° = cos60° = 1/2   cos30° = sin60° = √3/2

tan30° = √3/3   tan45° = 1   tan60° = √3

⚠️ 易混概念辨析

频率 vs 概率

频率：客观数据（试验次数/发生次数），可变化

概率：理论值，固定不变（抛硬币正面概率 = 0.5）

大量重复试验时，频率稳定在概率附近

全等 vs 相似

全等(≅)：形状相同 + 大小相等，所有边角完全相同

相似(∼)：形状相同，大小可不同，对应边成比例、对应角相等

面积比 = 相似比的平方

一次函数 k、b 的意义

y = kx + b：k ≠ 0；k > 0 过一三象限，k < 0 过二四象限；|k| 越大直线越陡；b 是截距

📝 典型例题

例1（一元二次方程）

方程 x² - 5x + k = 0 有两个不相等实根，求 k 的取值范围。

解析：Δ = 25 - 4k > 0 → k < 25/4 → k < 6.25

例2（圆，垂径定理）

半径为5，弦AB长为8，求圆心到弦AB的距离。

解析：r² = (d/2)² + d距² → 25 = 16 + d距² → d距² = 9
答案：3
