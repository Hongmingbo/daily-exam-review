📌 考点一：函数三剑客（一次·二次·反比例）

① 一次函数 y = kx + b（k ≠ 0）

图象：直线，k 决定倾斜方向（k > 0 左到右上升，k < 0 下降），b 决定与 y 轴交点

中考重点：求解析式、判断图象走势、求坐标轴交点、实际应用题

⚠️ 易错：正比例函数是特殊的一次函数（b = 0）；混淆 k/b 符号与图象对应关系
② 二次函数 y = ax² + bx + c（a ≠ 0）

图象：抛物线，三大要素：开口方向（a > 0 向上有最小值，a < 0 向下有最大值）、对称轴（x = −b/2a）、顶点

顶点坐标：(−b/2a, (4ac−b²)/4a)

中考重点：画图象、求解析式、分析性质、最值问题、与几何综合

⚠️ 易错：a 决定开口大小而非方向；与一次函数联立求解交点时漏解
③ 反比例函数 y = k/x（k ≠ 0）

图象：双曲线，关于原点对称，不与坐标轴相交

所在象限：k > 0 在一、三象限；k < 0 在二、四象限

中考重点：求 k 值、判断象限、比较函数值大小、与一次函数综合交点问题

⚠️ 易错：忽略 x ≠ 0 的取值范围；跨象限判断增减性时混淆与一次函数的性质

📌 考点二：圆上点到定点距离的最值问题

核心思路：数形结合（圆的几何性质 + 两点间距离公式）

第一步：分析动点轨迹，确定圆心 C(a, b) 和半径 r

第二步：将目标表达式转化为动点到定点 P 的距离

第三步：利用圆的性质——圆外一点到圆上点的最大/最小距离 = 该点到圆心的距离 ± 圆的半径

具体步骤

步骤1：由方程 (x−a)² + (y−b)² = r² 确定圆心 C(a, b)，半径 r

步骤2：求圆心 C 到定点 P(x₀, y₀) 的距离 d = √[(x₀−a)² + (y₀−b)²]

步骤3：最大距离 = d + r；最小距离 = |d − r|（若 d > r）；若 d = r 则最小距离为 0

举例：已知圆 x² + y² = 4，求圆上点到原点距离的最值

圆心 C(0, 0)，半径 r = 2；原点 O 到 C 距离 d = 0

最大距离 = 0 + 2 = 2；最小距离 = |0 − 2| = 2（此时点 P 为圆上任意点，到原点距离均为 2）

⚠️ 易错：求距离平方最值时，先求距离最值再平方，不要直接对 d ± r 求平方！

📌 考点三：勾股定理

勾股定理：直角三角形中，a² + b² = c²（a、b 为直角边，c 为斜边）

逆定理：若三边满足 a² + b² = c²，则为直角三角形

常用勾股数：3-4-5、5-12-13、7-24-25、8-15-17、9-40-41

⚠️ 易错：判断直角三角形时不要忘记用逆定理验证；题目没说直角时先证明直角再用勾股
