📌 考点一：函数三剑客

① 一次函数 y = kx + b（k ≠ 0）

图象：直线，k 决定倾斜方向（k > 0 从左到右上升，k < 0 下降），b 决定与 y 轴交点

待定系数法：一设（设 y = kx + b）、二代（代入两点坐标）、三列（列方程组）、四还原

中考重点：求解析式、判断图象走势、求坐标轴交点、实际应用题

⚠️ 易错：正比例函数是特殊的一次函数（b = 0）；混淆 k/b 符号与图象对应关系
② 二次函数 y = ax² + bx + c（a ≠ 0）

顶点坐标：(−b/2a, (4ac−b²)/4a)，用配方法求顶点式

图象三要素：开口方向（a > 0 向上有最小值，a < 0 向下有最大值）；对称轴 x = −b/2a；顶点

平移规律：平移必须先化成顶点式 y = a(x−h)² + k，遵循"左加右减、上加下减"

中考重点：用配方法求顶点坐标；二次函数与几何综合题；最值问题

⚠️ 易错：平移题忘化成顶点式；与一次函数联立求交点时漏解
③ 反比例函数 y = k/x（k ≠ 0）

图象：双曲线，关于原点对称，不与坐标轴相交

所在象限：k > 0 在一、三象限；k < 0 在二、四象限

中考重点：求 k 值、判断象限、比较函数值大小、与一次函数综合交点问题

⚠️ 易错：忽略 x ≠ 0 的取值范围；跨象限判断增减性时与一次函数性质混淆

📌 考点二：锐角三角函数

特殊角三角比（必须背熟）
角度 sin α cos α tan α ctg α
30° ½ √3/2 √3/3 √3
45° √2/2 √2/2 1 1
60° √3/2 ½ √3 √3/3

📌 考点三：圆的重要定理

① 垂径定理（核心重点）

垂直于弦的直径平分这条弦，并且平分弦所对的两条弧

推论：弦的垂直平分线经过圆心；圆心到弦的垂直距离 d = √(R²−(L/2)²)

② 圆心角、弧、弦、弦心距关系定理

在同圆或等圆中，相等的圆心角所对的弧相等，所对的弦相等，所对的弦心距相等

记忆口诀：同圆等圆中，"弧角弦距"四等——弧等则角等，角等则弦等，弦等则距等

③ 直线与圆的位置关系

d > R：相离（无交点）　d = R：相切（1个交点）　d < R：相交（2个交点）

切线的判定：经过半径的外端并且垂直于这条半径的直线是圆的切线

切线的性质：切线垂直于经过切点的半径

⚠️ 易错：切线的判定和性质定理容易混淆——判定是"垂直半径"，性质是"半径垂直"
