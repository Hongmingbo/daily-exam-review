📖 基础知识 · 酸碱盐专题

📌 酸碱盐知识体系（中考重点）

① 酸的化学性质（5条）

  • 酸 + 金属氧化物 → 盐 + 水
    例：Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O
  • 酸 + 金属 → 盐 + 氢气↑（金属活动性顺序中H之前的金属）
    例：Zn + H₂SO₄ → ZnSO₄ + H₂↑
  • 酸 + 碱 → 盐 + 水（中和反应）
  • 酸 + 盐 → 新酸 + 新盐（符合复分解条件）
  • 酸 + 碳酸盐 → 盐 + 水 + CO₂↑

② 碱的化学性质（4条）

  • 碱 + 非金属氧化物 → 盐 + 水
    例：2NaOH + CO₂ → Na₂CO₃ + H₂O（NaOH必须密封保存！）
  • 碱 + 酸 → 盐 + 水（中和反应）
  • 碱 + 盐 → 新碱 + 新盐（两者都需可溶，且生成物有沉淀）
  • 碱溶液与指示剂反应：石蕊遇碱变蓝，酚酞遇碱变红

③ 盐的化学性质（4条）

  • 盐 + 金属 → 新金属 + 新盐（金属活动性顺序中前面的金属置换后面的）
    例：Fe + CuSO₄ → FeSO₄ + Cu（湿法炼铜）
  • 盐 + 酸 → 新盐 + 新酸
  • 盐 + 碱 → 新盐 + 新碱
  • 盐 + 盐 → 两种新盐（两者都溶，生成物有沉淀）

④ 复分解反应条件（口诀：生成气体/沉淀/水）

  • 两种化合物互相交换成分，生成物中有沉淀、气体或水
  • 酸 + 碱：一定发生（生成水）
  • 酸 + 碳酸盐：一定发生（生成CO₂气体）
  • 盐 + 碱：反应物都必须溶于水
  • 盐 + 盐：反应物都必须溶于水
重点：判断复分解反应能否发生 = 看生成物是否有沉淀/气体/水

📊 化学计算题核心技巧

① 质量守恒定律

反应物总质量 = 生成物总质量

利用：原子种类、数目、质量不变

题型：求某反应物/生成物的质量、求化学式、求某元素质量分数

② 差量法（中考高频）

差量 = 反应前物质总质量 − 反应后物质总质量 = 气体质量 = 沉淀质量

例：CaCO₃高温分解，差量 = CO₂质量

CaCO₃ → CaO + CO₂↑

100g.......................44g

差量：每分解100g CaCO₃，固体质量减少44g

③ 化学方程式计算步骤

设（未知数）→ 写（化学方程式）→ 标（已知量+关系量）→ 列（比例式）→ 求（计算）→ 答

💡 注意：化学方程式必须配平！相对分子质量必须算准！

④ 溶质质量分数计算

ω = m溶质 / m溶液 × 100%

m溶液 = m溶质 + m溶剂 = ρ × V溶液

稀释问题：m浓溶液 × ω浓 = m稀溶液 × ω稀

注意：稀释问题中，溶剂质量守恒，但溶液质量不守恒！

🔬 中考化学实验操作要点

① 粗盐提纯（中考高频实验）

溶解：玻璃棒搅拌，加速溶解

过滤：一贴二低三靠；滤纸紧贴漏斗壁，液面低于滤纸边缘，玻璃棒靠在三层滤纸处

蒸发：玻璃棒搅拌（防止局部过热），当出现较多晶体时停止加热，利用余热蒸干

滤液浑浊原因：滤纸破损、液面高于滤纸边缘、仪器不干净

② 氧气实验室制取

2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（加热高锰酸钾）

2KClO₃ → 2KCl + 3O₂↑（二氧化锰催化氯酸钾）

收集方法：向上排空气法（O₂密度大于空气）或排水法（O₂不易溶于水）

验满：用带火星木条放在集气瓶口，复燃则满

③ 二氧化碳实验室制取

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

长颈漏斗必须液封，防止气体从漏斗逸出

收集方法：向上排空气法（CO₂密度大于空气，能溶于水）

验满：用燃着木条放在集气瓶口，熄灭则满

④ 金属活动性顺序实验

K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

金属 + 酸 → 氢气：只有在H之前的金属才能反应

金属 + 盐溶液 → 新金属 + 新盐：前面的金属置换后面的金属

注意：K、Ca、Na非常活泼，不能与盐溶液发生置换反应（先与水反应）

🎨 常见物质颜色与状态（推断题题眼）

固体颜色

  • 黑色：碳（C）、氧化铜（CuO）、四氧化三铁（Fe₃O₄）、二氧化锰（MnO₂）、铁粉（Fe）
  • 红色：铜（Cu）、红磷（P）、氧化铁（Fe₂O₃）
  • 绿色：碱式碳酸铜（Cu₂(OH)₂CO₃，铜绿）
  • 蓝色：胆矾（CuSO₄·5H₂O）、氢氧化铜（Cu(OH)₂）
  • 白色：大多数盐和碱（除带颜色外）

溶液颜色

  • 蓝色：Cu²⁺溶液（CuSO₄溶液等）
  • 黄色：Fe³⁺溶液（FeCl₃溶液、Fe₂(SO₄)₃溶液）
  • 浅绿色：Fe²⁺溶液（FeSO₄溶液、FeCl₂溶液）
  • 无色：大多数盐溶液、酸溶液、碱溶液

沉淀颜色

  • 蓝色沉淀：Cu(OH)₂
  • 红褐色沉淀：Fe(OH)₃
  • 白色沉淀：CaCO₃、BaCO₃、AgCl、BaSO₄、Mg(OH)₂、Zn(OH)₂
注意：Fe(OH)₂是白色沉淀，但极易被氧化成Fe(OH)₃（灰绿色→红褐色）

气体检验

  • O₂：带火星木条，复燃
  • CO₂：澄清石灰水，变浑浊；燃着木条，熄灭
  • H₂：点燃，发出淡蓝色火焰，产物只有水
  • CO：点燃，发出蓝色火焰，产物只有CO₂；通入灼热CuO，黑色变红色
  • NH₃：湿润红色石蕊试纸，变蓝；有刺激性气味

💡 复习建议

  1. 酸碱盐是中考核心：背熟通性，理解反应条件，能写化学方程式
  2. 化学方程式每天默写10个：重点练配平，条件、↑↓符号要完整
  3. 推断题找"题眼"：特殊颜色（蓝色沉淀、红色固体）、气体、沉淀、反应条件
  4. 计算题掌握质量守恒和差量法：多练步骤书写，规范表达
  5. 实验题注意操作规范：现象描述、结论得出要准确
