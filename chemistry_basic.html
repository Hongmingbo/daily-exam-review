📌 易错化学式和概念（10个）

① 常见化学式易错

  • 碳酸钠：Na₂CO₃（不是NaCO₃！）
  • 氢氧化钙：Ca(OH)₂（石灰水，不是CaOH！）
  • 硫酸铜：CuSO₄（蓝色，≠ 硫酸亚铁FeSO₄绿色）
  • 硝酸铵：NH₄NO₃（两种氮元素，易漏！）
  • 高锰酸钾：KMnO₄（≠ K₂MnO₄锰酸钾）

② 反应类型易混淆

  • 燃烧 ≠ 氧化：燃烧是剧烈氧化反应（发光放热）；缓慢氧化（如铁生锈）也是氧化，但不发光
  • 复分解反应条件：生成物必须有沉淀、气体或水
    · 酸 + 碱 → 盐 + 水（一定发生！）
    · 盐 + 盐 → 两种新盐（需两者都溶于水！）

③ 溶液易错概念

  • 溶质质量分数：ω = m溶质 / m溶液 × 100%
    ≠ m溶质 / m溶剂 ！（分母是溶液不是溶剂！）
  • 饱和溶液 ≠ 浓溶液：
    · 饱和：不能再溶解更多该溶质
    · 浓：溶质质量分数大
    · 饱和溶液不一定是浓溶液（如Ca(OH)₂饱和溶液很稀）

④ 常见实验现象

  • 铁丝在氧气中燃烧：火星四射，生成黑色固体（Fe₃O₄），不是Fe₂O₃！
    氢气燃烧：淡蓝色火焰，不是蓝色！
    CO₂通入澄清石灰水：先变浑浊，后变澄清（CO₂过量）
    铁与硫酸铜：铁表面覆盖红色铜（湿法炼铜）
    氧化钙加水：放出大量热，生成熟石灰Ca(OH)₂

⚗️ 必背化学方程式

① 燃烧反应
C + O₂ → CO₂（点燃）
2H₂ + O₂ → 2H₂O（点燃）
CH₄ + 2O₂ → CO₂ + 2H₂O（点燃）
② 还原反应
2CuO + C → 2Cu + CO₂↑（高温）
Fe₂O₃ + 3CO → 2Fe + 3CO₂（高温）
③ 酸碱中和
NaOH + HCl → NaCl + H₂O
2NaOH + H₂SO₄ → Na₂SO₄ + 2H₂O
Ca(OH)₂ + 2HCl → CaCl₂ + 2H₂O
④ 金属与酸反应（实验室制氢气）
Zn + H₂SO₄ → ZnSO₄ + H₂↑
Fe + 2HCl → FeCl₂ + H₂↑
💡 配平技巧：观察法 → 找最复杂化合物 → 依次配平原子数
例：Fe + O₂ → Fe₃O₄ → 先配Fe：3Fe + O₂ → Fe₃O₄ → 再配O：3Fe + 2O₂ → Fe₃O₄ ✓

🎨 物质颜色与俗称

物质俗称颜色/特征
Fe₂O₃赤铁矿/铁红红色粉末
Fe₃O₄磁铁矿黑色晶体
CuSO₄胆矾/蓝矾白色→蓝色（加水）
CaO生石灰白色固体
Ca(OH)₂熟石灰/消石灰白色粉末，微溶
CaCO₃石灰石/大理石白色固体，难溶
NaCl食盐白色晶体
KMnO₄高锰酸钾/灰锰氧紫黑色晶体
Cu₂(OH)₂CO₃铜绿/孔雀石绿色粉末
📝 记忆口诀：铜绿出铁锈，氨气有刺激。
铁生锈：Fe + O₂ + H₂O → Fe₂O₃·nH₂O（缓慢氧化）

🔬 重要实验操作要点

① 氧气制备（实验室）

2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（加热）

· 试管口略向下倾斜（防止水倒流）

· 铁夹夹在试管口约1/3处

· 用排水法收集时，等气泡均匀冒出再收集（否则混有空气）

· 先撤导管再移酒精灯（防止水倒流炸裂试管）

② 电解水实验

2H₂O → 2H₂↑ + O₂↑（通电）

· 正氧负财富：电源正极产O₂，负极产H₂

· 体积比：H₂ : O₂ = 2 : 1

· 质量比：H₂ : O₂ = 1 : 8（水中氢氧元素质量比固定）

③ 粗盐提纯

溶解 → 过滤（去泥沙）→ 蒸发结晶

蒸发时不能把水蒸干再停止加热！ 当有较多晶体析出时即停止，利用余热蒸干。 目的：防止温度过高导致晶体分解或飞溅。

💡 复习建议

  1. 每天背诵5个化学方程式：重点练配平，条件、箭头要写完整
  2. 易错化学式要手写练习：特别小心下标数字（如Na₂CO₃的₂）
  3. 实验操作题：熟记"先撤导管后移灯"等规范步骤
  4. 溶液计算：分清溶质/溶剂/溶液，质量分数公式要理解不要死背
