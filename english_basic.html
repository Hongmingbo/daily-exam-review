📖 词汇语法 · 英语

📝 中考高频词汇（20组）

① 动词短语高频考点

| 短语 | 含义 |

| look forward to doing | 期待、盼望做某事 |

| take part in / join in | 参加（活动/组织） |

| take care of | 照顾、照料 |

| put off / put away / put up | 推迟 / 收起 / 搭建 |

| run out of / run into | 用完 / 撞上 |

| come across / come up with | 偶然发现 / 想出 |

| get along/on with | 与……相处融洽 |

② 形近词/易混词辨析

alone / lonely：alone 客观表示"独自一人"，lonely 强调"孤独寂寞感"

except / besides / except for：except 除……之外（不包含），besides 除……之外（还包含），except for 整体与部分的差异

spend / cost / take / pay：
· sb. spend time/money on sth.
· Sth. cost sb. money（无被动）
· It takes sb. time to do.
· sb. pay money for sth.

receive / accept：receive 表示"收到"，accept 表示"接受"（认可）

wear / dress / put on：wear 穿（状态）/ dress 穿（动作，后接人）/ put on 穿上（短暂动作）

③ 重点词汇用法

enough：adj. + enough + to do / n. + enough
→ He is tall enough to reach the book.

so...that / such...that：
· so + adj./adv. + that
· such + (a/an + adj. + n.) + that
→ It is such an interesting book that everyone likes it.

used to / be used to doing：used to 过去常常（现在不做了）/ be used to doing 习惯于做某事

keep / stop / prevent...from：三者都可表示"阻止"，from 不能省略（keep from / stop from / prevent from）

④ 介词搭配高频

in：in the morning/afternoon/evening | in May | in 2024 | in spring

on：on Monday | on the morning of July 1st | on the wall | on weekdays

at：at 7 o'clock | at noon/night | at the age of 15 | at school/work

by：by bus/train/plane | by the river | by mistake | by the time (that)

⑤ 核心词汇辨义

noise / voice / sound：sound 泛指声音 / voice 人的嗓音 / noise 噪音

problem / question / matter：problem 难题（需解决）/ question 提问（需回答）/ matter 事情（需处理）

journey / trip / travel / tour：journey 长途旅行 / trip 短途往返 / travel 旅行（总称）/ tour 观光旅行

common / usual / ordinary：common 常见的（强调普遍性）/ usual 惯常的（强调习惯）/ ordinary 平凡的（强调普通）

📌 重点语法（5大高频考点）

① 定语从句（中考高频）

结构：名词 + 关系代词/副词 + 从句

关系代词：who（代人）/ whom（代人，宾格）/ whose（所有格）/ which（代物）/ that（可代人和物）

关系副词：when（时间）/ where（地点）/ why（原因）

例：The book which/that I bought yesterday is interesting.
This is the factory where my father works.
I still remember the day when we first met.

⚠️ 易错：先行词为不定式/最高级/序数词时只用 that；关系代词在定语从句中作宾语时可省略。

② 被动语态

结构：be + 过去分词（p.p.）

各时态形式：

· 一般现在时：am/is/are + p.p.
· 一般过去时：was/were + p.p.
· 一般将来时：will be + p.p. / is going to be + p.p.
· 现在完成时：have/has been + p.p.
· 情态动词：can/must/may be + p.p.

主动改被动：宾语 → 主语 + be + p.p. + (by + 执行者)

③ 宾语从句（三大考点）

1. 语序：陈述句语序（主语在前）
→ I think that he is right.（√）
→ I think that is he right.（✗）

2. 时态呼应：主句过去时 → 从句过去相应时态
→ He said that he liked music.（一般过去）
→ He said that the earth is round.（客观真理仍用现在）

3. 引导词：that（无意义）/ whether/if（是否）/ 疑问词（who/what/where...）

④ 条件状语从句（if/unless）

if 从句：主将从现（主句将来时，从句现在时）
→ If it rains tomorrow, I will stay at home.

unless = if not：unless 除非（表负面条件）
→ You will fail unless you work harder.

时间顺序：主句用将来时，从句用现在时代替将来时

⑤ 感叹句与反意疑问句

感叹句：
· What + (a/an) + adj. + n. + (主谓)!
→ What a beautiful flower (it is)!
· How + adj./adv. + (主谓)!
→ How beautiful the flower is!

反意疑问句：前肯后否，前否后肯
→ He is a teacher, isn't he?
→ You don't like it, do you?

注意：let's 开头 → shall we？ / let us 开头 → will you？

📖 阅读理解解题技巧

💡 四步法：①看题目划关键词 → ②带着问题读文章 → ③定位原文关键句 → ④比对选项选答案

细节题：在原文中直接找答案，注意同义替换（most≈almost / however≈but）

推理题：不能选原文直接陈述的选项，需推断作者意图或文章隐含意思

主旨题：看首段和末段，尤其注意每段首句，往往是主题句

猜词题：根据上下文语境（and/but/because 等逻辑词）推断生词含义

态度题：positive（积极）/ negative（消极）/ neutral（中立）

💡 复习建议

  1. 词汇每天背20个：重点背动词短语（look forward to, take part in 等）、介词搭配和形近词辨析
  2. 语法专项突破：定语从句、被动语态、宾语从句每天各练3题，注意时态和语态的结合
  3. 阅读每天1篇：先看题目再读文章，带着问题找答案，注意找同义替换
  4. 作文模板：准备开头句型（Recently.../With the development of...）+ 3个万能中间段结构
  5. 听力训练：每天听一套听力题，跟读原文，纠正发音，培养语感
