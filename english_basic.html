📖 词汇语法 · 英语

📝 中考高频词汇（20组）

① 动词短语高频考点

| 短语 | 含义 |

| look forward to | 期待，盼望 |

| take part in | 参加（活动） |

| take care of | 照顾，照料 |

| put off | 推迟，拖延 |

| put up with | 忍受，容忍 |

| run out of | 用完，耗尽 |

| come up with | 想出，提出（主意） |

| get along/on with | 与……相处融洽 |

② 形近词辨析

alone / lonely：alone 客观表示"独自"，lonely 强调"孤独寂寞感"

except / besides：except 除……之外（不包括），besides 除……之外（还包括）

spend / cost / take / pay：
· sb. spend time/money on sth.
· Sth. cost sb. money（无被动）
· It takes sb. time to do.
· sb. pay money for sth.

③ 重点词汇用法

enough：adj. + enough + to do / n. + enough
→ He is tall enough to reach the book.（够高去做某事）

so...that / such...that：
· so + adj./adv. + that
· such + (a/an + adj. + n.) + that
→ It is such an interesting book that everyone likes it.

used to：过去常常（现在不再）
→ I used to play football after school.（过去习惯）

📌 重点语法（5大高频考点）

① 定语从句（中考高频）

结构：名词 + 关系代词/副词 + 从句

关系代词：who（代人）/ whom（代人，宾格）/ whose（所有格）/ which（代物）/ that（可代人和物）

关系副词：when（时间）/ where（地点）/ why（原因）

例：The book which/that I bought yesterday is interesting.
This is the factory where my father works.
I still remember the day when we first met.

⚠️ 易错：先行词为不定式/最高级/序数词时只用 that；关系代词在定语从句中作宾语时可省略。

② 被动语态

结构：be + 过去分词（p.p.）

各时态形式：

· 一般现在时：am/is/are + p.p.
· 一般过去时：was/were + p.p.
· 一般将来时：will be + p.p. / is going to be + p.p.
· 现在完成时：have/has been + p.p.
· 情态动词：can/must/may be + p.p.

主动改被动：宾语 → 主语 + be + p.p. + (by + 执行者)

③ 宾语从句（三大考点）

1. 语序：陈述句语序（主语在前）
→ I think that he is right.（√）
→ I think that is he right.（✗）

2. 时态呼应：主句过去时 → 从句过去相应时态
→ He said that he liked music.（一般过去）
→ He said that the earth is round.（客观真理仍用现在）

3. 引导词：that（无意义）/ whether/if（是否）/ 疑问词（who/what/where...）

📖 完形填空解题技巧

💡 三步法：①通读全文把握大意 → ②逐句分析选择答案 → ③复读检查确认逻辑

上下文线索：注意 but/and/so/because 等逻辑词；空前后的同义词/反义词提示

固定搭配：look forward to doing / be used to doing / spend...doing

情感色彩：positive（积极）/ negative（消极）/ neutral（中性）

词性判断：冠词后填名词/形容词；动词前填副词；主语位置填名词/代词

💡 复习建议

  1. 词汇每天背20个：重点背动词短语（look forward to, take part in 等）和形近词辨析
  2. 语法专项突破：定语从句、被动语态、宾语从句每天各练3题
  3. 阅读每天1篇：先看题目再读文章，带着问题找答案
  4. 作文模板：准备开头句型（Recently.../With the development of...）+ 3个万能中间段
