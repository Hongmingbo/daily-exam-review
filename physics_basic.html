📌 力学核心公式

① 速度公式
v = s / t

适用条件：匀速直线运动

单位：v—m/s，s—m，t—s

注意：全程平均速度 ≠ 各段速度平均值；1 m/s = 3.6 km/h

② 密度公式
ρ = m / V

适用条件：同种物质的均匀物体

单位：ρ—kg/m³，m—kg，V—m³

注意：密度是物质特性，与 m、V 无关；水 ρ = 1.0 × 10³ kg/m³

③ 压强公式
p = F / S 　　（固体压强）
p = ρgh 　　（液体压强）

固体压强：计算柱形固体对水平面的压强

液体压强：液体内部压强，h 为该点到液面的深度

易错：液体压强只与 ρ液、h 有关，与液体重力、容器形状无关！

④ 浮力公式（4种方法）
F浮 = G物 − F拉 　　（称重法）
F浮 = ρ液gV排 　　（阿基米德原理）
F浮 = F向上 − F向下 　　（压力差法）
F浮 = G物 　　（漂浮/悬浮时）

适用条件：V排 ≤ V物；物体完全浸没时 V排 = V物

易错：物体浮沉条件：ρ物 > ρ液 → 下沉；ρ物 = ρ液 → 悬浮；ρ物 < ρ液 → 上浮

⑤ 机械效率
η = W有用 / W总 × 100%
W总 = W有用 + W额外

滑轮组：η = Gh / (Fnh) = G / (nF)（忽略摩擦时）

斜面：η = Gh / (FL)

易错：有用功 ≠ 总功，机械效率永远小于 100%

⑥ 功和功率
W = Fs 　　（功的定义）
P = W / t = Fv 　　（功率）

注意：做功的两个必要条件：有力 F，有距离 s（同方向）

易错：克服重力做功时 W = Gh；克服摩擦做功时 W = fs（摩擦力乘路程）

📌 电学核心公式

① 欧姆定律
I = U / R
U = IR 　　R = U / I

适用条件：纯电阻电路

注意：电阻是导体特性，与 U、I 无关（同种材料、长度、横截面积）

② 串并联电路特点
串联：R总 = R₁ + R₂ + … 　　I相等　U分压
并联：1/R总 = 1/R₁ + 1/R₂ + … 　　U相等　I分流

分压公式（串联）：U₁/U₂ = R₁/R₂

分流公式（并联）：I₁/I₂ = R₂/R₁

③ 电功与电功率
W = UIt = I²Rt = U²t/R
P = W/t = UI = I²R = U²/R

适用条件：所有电路（总公式）；后两个仅适用于纯电阻

易错：实际功率不是额定功率，注意区分！

电能表：1 kW·h = 3.6 × 10⁶ J

④ 焦耳定律
Q = I²Rt

适用条件：所有电路的热计算

易错：纯电阻电路 Q = W = Pt，非纯电阻电路 Q < W

📌 热学核心公式

① 吸热与放热
Q吸 = cm(t₂ − t₁)
Q放 = cm(t₁ − t₂)

适用条件：无物态变化时

单位：c—J/(kg·℃)，m—kg，t—℃

② 热值（燃料燃烧）
Q放 = mq 　　（固体/液体）
Q放 = Vq 　　（气体燃料）

注意：热值是燃料特性，与燃料质量无关

③ 热效率
η = Q吸 / Q放 × 100%

易错：比热容是物质特性，与质量、体积无关！

📌 光学核心

① 光的反射定律

反射光线、入射光线、法线在同一平面内

反射光线、入射光线分居法线两侧

反射角 = 入射角

特殊情况：当光线垂直入射时，入射角 = 反射角 = 0°

② 光的折射规律

折射光线、入射光线、法线在同一平面内

折射光线、入射光线分居法线两侧

光从空气斜射入水/玻璃：折射角 < 入射角

光从水/玻璃斜射入空气：折射角 > 入射角

口诀：空气中角度大（无论入射还是折射）

③ 凸透镜成像规律
1/u + 1/v = 1/f

u > 2f：倒立缩小实像（照相机）

f < u < 2f：倒立放大实像（投影仪）

u < f：正立放大虚像（放大镜）

u = 2f：倒立等大实像

口诀：一倍焦距分虚实，二倍焦距分大小

📌 电磁学核心

① 磁场对电流的作用
F = BIL 　　（通电导线在磁场中受力的力）

左手定则：伸开左手，使拇指与四指垂直并在同一平面内，让磁感线穿入手心，四指指向电流方向，则拇指所指方向为导体受力方向

② 电磁感应
E = nΔΦ/Δt 　　（感应电动势）

右手定则：伸开右手，使拇指与四指垂直并在同一平面内，让磁感线穿入手心，拇指指向导体运动方向，则四指所指方向为感应电流方向

口诀：左力右电（左手定则用于受力/电动机，右手定则用于感应电流/发电机）

③ 家庭电路

火线（220V）+ 零线（0V）+ 地线（保护）

保险丝接在火线上；插座左零右火上地

安全用电：有金属外壳的家用电器必须接地
